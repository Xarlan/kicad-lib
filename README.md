# kicad-lib
This repository contain additional symbols and footprints for KiCad

## symbols:
* LDO
    - AMS1117
* Chip
    - MAX3421E (USB 2.0 <-> SPI)
* Connector
    - USB type A
    - micro_USB
* Isolator
	- ADUM4160 (isolator for USB)
* Triac
	- Z0103MN 5AA4, 1А, 600В [SOT-223]
   
## footprints:
* HC-49SM
* TQPF32 size 5x5mm e=0.5
* USBA_1J_DS_1095_WR
* micro_USB_KLS1-233-0-0-1-T
* KLS2-126-5.00
